# `jar4 setup`

Initialize `~/.jar4/jar4.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
jar4 setup
jar4 setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
jar4 setup --wizard
```
