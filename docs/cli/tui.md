# `jar4 tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
jar4 tui
jar4 tui --url ws://127.0.0.1:18789 --token <token>
jar4 tui --session main --deliver
```
